将hexo博客迁移到github

Author: JTongChen   Date: August 17, 2019  18:27:02   Pageviews: 0

很长时间没有动hexo博客，于是最近想重新拾起，并且把博客迁移到github上，不再使用开源中国的部署，并且为博客更换主题。下面介绍一下大致的迁移流程。

流程：

  1. 首先，需要注册个github账号，并且在github上新建仓库，命名为:你的guthib帐户名.github.io（注意：项目应该设置为共有public权限，否则访问页面失败。）
  2. 下载git客户端，设置ssh key将客户端与github账户绑定(具体步骤看git ssh配置)
  1. 安装hexo，具体步骤在之前的安装hexo并部署到oschina里面已经描述清楚。
